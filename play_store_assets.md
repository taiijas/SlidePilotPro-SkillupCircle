## 2. Full Description (Play Store)

Transform your Android device into a professional-grade Bluetooth presentation remote and wireless trackpad with **SlidePilot Pro**. 

Control Microsoft PowerPoint, Google Slides, Apple Keynote, PDFs, and media players seamlessly on macOS, Windows, and Linux—**without installing any companion software, drivers, web extensions, or server applications on your computer.**

SlidePilot Pro leverages native Android Bluetooth HID (Human Interface Device) profile features to emulate a physical hardware keyboard and mouse over Bluetooth.

### KEY FEATURES
- **Zero-Software Setup**: Connects directly via your device’s native Bluetooth settings. Zero setup required on the host computer.
- **Presenter Mode**: 
  - Supports PowerPoint, Keynote, Google Slides, and PDF navigation.
  - Large next/prev slide touch targets and swipe gestures.
  - Quick keys for F5 (Start Show), Esc (Exit Show), B (Black Screen), and W (White Screen).
  - Customizable profile mapping based on OS and application.
- **Wireless Mouse & Trackpad**:
  - Smooth mouse cursor movement with adjustable sensitivity.
  - Mouse clicks (Left Click, Right Click, Middle Click).
  - Two-finger vertical scrolling.
- **App Lifecycle & Auto-Reconnect**:
  - Automatically restores connection and re-registers the Bluetooth service when the app is resumed.
- **Haptic Feedback**: Subtle vibration on clicks/gestures for physical control confirmation.
- **Release Safety Boundaries**: Hardened error handling to ensure a crash-free experience.

### COMPATIBILITY
- **Android Device**: Requires Android 11 (API 30) or higher with Bluetooth HID Device profile support (enabled on most modern phones).
- **Host Computers**: Compatiable with macOS, Windows, Linux, ChromeOS, and iPadOS.
